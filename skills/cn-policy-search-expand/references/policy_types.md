# 中国政策类型扩展词列表

## 通用政策类型

以下政策类型适用于大多数省份和城市：

| 政策类型 | 说明 |
|---------|------|
| 财政政策 | 财政支出、预算管理、政府采购等 |
| 产业政策 | 产业发展规划、产业升级、产业链等 |
| 人才政策 | 人才引进、人才补贴、人才落户等 |
| 补贴政策 | 企业补贴、项目补贴、专项补贴等 |
| 税收优惠政策 | 减税降费、税收减免、优惠税率等 |
| 农业政策 | 乡村振兴、农业补贴、农村发展等 |
| 工业政策 | 工业发展、制造业、园区政策等 |
| 科技创新政策 | 科技创新、研发补贴、高新技术等 |
| 中小企业政策 | 中小企业扶持、融资担保、市场准入等 |
| 对外贸易政策 | 进出口、外贸补贴、跨境电商等 |

## 服务业政策

| 政策类型 | 说明 |
|---------|------|
| 服务业政策 | 现代服务业、商贸流通、消费补贴等 |
| 数字经济政策 | 数字产业、数字化转型、数据要素等 |
| 文化旅游政策 | 文旅融合、旅游补贴、文化产业等 |
| 养老服务政策 | 养老产业、居家养老、康养等 |

## 区域发展政策

| 政策类型 | 说明 |
|---------|------|
| 区域发展政策 | 区域协调、结对帮扶、园区共建等 |
| 城市规划政策 | 城市建设、房地产、城市更新等 |
| 环保政策 | 生态保护、绿色发展、节能减排等 |

## 特定城市附加

### 直辖市
- 北京、上海、天津、重庆

### 计划单列市
- 深圳、青岛、宁波、厦门、大连

### 省会城市
- 成都、杭州、广州、武汉、西安、南京、长沙、郑州、济南等

这些城市可附加特定政策类型，如：
- 深圳：先行示范区政策、粤港澳大湾区政策
- 上海：自贸区政策、浦东政策
- 北京：首都功能政策、副中心政策
