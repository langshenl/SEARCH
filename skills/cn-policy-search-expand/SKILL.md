# 中国政策搜索扩展

## 触发条件

当用户搜索词**同时满足**以下条件时，触发政策扩展：

1. 包含省份或城市名称（湖北、上海、北京、广东、深圳等）
2. 包含"政策"一词
3. 包含年份（阿拉伯数字，如2026、2025）

**示例：**
- ✅ "2026年湖北政策" → 触发扩展
- ✅ "2025年上海政策" → 触发扩展
- ❌ "新能源汽车政策" → 不触发（无省份/城市 + 无年份）
- ❌ "人工智能政策" → 不触发（无省份/城市）

## 扩展流程

### 第一步：解析搜索词

从搜索词中提取：
- **省份/城市名**：如"湖北"、"上海"
- **年份**：如"2026年"、"2025年"
- **原始词干**：`省份 + 年份 + "政策"`

### 第二步：加载扩展词

读取 `references/policy_types.md` 获取政策类型扩展词列表。

### 第三步：生成扩展搜索词

将年份 + 省份/城市 + 每个政策类型组合，生成多个搜索词：

```
原始：2026年湖北政策
扩展：
- 2026年湖北财政政策
- 2026年湖北产业政策
- 2026年湖北人才政策
- 2026年湖北补贴政策
- 2026年湖北税收优惠政策
- 2026年湖北农业政策
- 2026年湖北工业政策
- 2026年湖北科技创新政策
- 2026年湖北中小企业政策
- 2026年湖北对外贸易政策
```

### 第四步：输出扩展结果

直接输出扩展后的搜索词列表（无需用户确认），然后使用 web_search 工具批量搜索。

### 第五步：合并去重

将所有搜索结果合并，按相关度/时间排序后输出。

## 时间词转换

用户输入"今年"、"去年"等相对年份时，自动转换为实际年份：
- 今年 → 2026
- 去年 → 2025

## 不触发场景

以下情况不触发扩展，直接按原词搜索：
- 搜索词已包含具体政策类型（如"湖北产业政策"）
- 搜索词无省份/城市信息（如"新能源汽车政策"）
- 搜索词无年份（如"湖北政策"）
