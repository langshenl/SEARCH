# smart-policy-search 技能安装说明

## 前置要求

### 1. Chrome 远程调试开启（必须）

1. 关闭所有 Chrome 窗口
2. 打开 Chrome，输入地址：`chrome://inspect/#remote-debugging`
3. 勾选 **"Allow remote debugging for this browser instance"**
4. 如果没有这个选项，需要用命令行启动：
   ```bash
   open -a "Google Chrome" --args --remote-debugging-port=9222 --no-first-run --no-default-browser-check --user-data-dir=/tmp/chrome-debug
   ```

### 2. CDP Proxy 服务（必须）

CDP Proxy 由 openclaw 的 web-access 技能管理。

检查是否正常：
```bash
curl http://localhost:3456/health
# 返回 {"status":"ok","connected":true} 表示正常
```

如果没有反应，在那台电脑的 openclaw 中检查 web-access 技能是否正常运行。

### 3. Python 依赖（必须）

```bash
pip3 install openpyxl
```

### 4. Exa API Key（可选，用于省份政策搜索）

- 前往 https://exa.ai 注册获取 API Key
- 设置环境变量：
  ```bash
  export EXA_API_KEY="your-api-key-here"
  ```
- 如果不设置，Exa 相关功能不可用，但部门官网搜索和CDP百度搜索仍可用

---

## 安装步骤

### 方法一：复制技能目录

1. 将整个 `smart-policy-search/` 文件夹复制到目标电脑的：
   ```
   ~/.openclaw/workspace-search/skills/smart-policy-search/
   ```

2. 重启 openclaw 使技能生效

### 方法二：仅复制脚本文件

```bash
# 复制脚本
cp smart-policy-search/scripts/unified_search.py ~/.openclaw/workspace-search/skills/smart-policy-search/scripts/unified_search.py
cp smart-policy-search/scripts/gov_config.py ~/.openclaw/workspace-search/skills/smart-policy-search/scripts/gov_config.py

# 复制文档
cp smart-policy-search/SKILL.md ~/.openclaw/workspace-search/skills/smart-policy-search/SKILL.md
```

## 验证是否安装成功

在 openclaw 中执行：
```
使用 smart-policy-search 技能搜索"武汉限行规定"
```

如果返回搜索结果，说明安装成功。

## 常见问题

**Q: curl localhost:3456/health 无响应**
A: CDP Proxy 服务未启动。在 openclaw 中检查 web-access 技能状态，或重启 openclaw。

**Q: "cdp_new 失败" 错误**
A: Chrome 远程调试未开启或端口不对。按第一步重新配置 Chrome。

**Q: Exa 搜索报 "API Key 未设置"**
A: 这是正常的 — Exa key 耗尽或不存在时，技能会自动降级到 CDP 百度搜索，不影响基本功能。

**Q: 部门官网搜索返回 0 条结果**
A: 该部门官网可能更改了搜索接口，技能会自动降级到 Exa 搜索（如果有 key）。
