# 官方站群搜索参考

> 用于 `cn-policy-multi-search`。当用户搜索某省/某市“所有政策”时，不要只用单一主政府官网，优先按“官方站群”思路扩展搜索。

## 使用方法

搜索某个地区时，至少按下面层级扩展：

1. 主政府官网
2. 办公厅 / 政府公报 / 政务公开目录
3. 重点厅局 / 委办局
4. 地市政府门户（省级搜索时）
5. 区级政府门户（城市级搜索时）
6. 官方政策平台 / 专题页（若存在）

如果某个具体域名不确定，也要按机构名称进行补搜，不要停留在单一主站。

## 使用提醒

- 这是搜索扩展参考，不要求每次都完整枚举全部站点。
- 但当结果明显偏少时，必须从这里继续扩展，而不是直接结束。
- 如果用户要“所有政策”，优先先补站群，再补搜索引擎。
