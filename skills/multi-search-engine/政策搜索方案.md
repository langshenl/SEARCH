# 国家政策搜索方案

## 概述

本方案用于搜索中国各省份政府网站上的国家政策内容，支持搜索 31 个省份/直辖市政府网站。

## 功能特点

- 🔍 **多省份覆盖**: 支持全部 31 个省份和直辖市政府网站
- 🔑 **双重搜索引擎**: 支持 Exa AI 和 Tavily API
- 📁 **结构化输出**: 结果保存为 JSON 和 Markdown 格式
- 📋 **规范格式**: 每条结果包含标题、正文、摘要、来源地方、原文地址

## 输出格式

每条搜索结果包含：

| 字段 | 说明 |
|------|------|
| 标题 | 文章标题 |
| 正文 | 文章完整正文（截取前2000字符） |
| 摘要 | AI 生成的摘要 |
| 来源地方 | 省份名称 |
| 原文地址 | 原始网页链接 |

## 使用方法

### 方法一：直接运行脚本

```bash
cd ~/.openclaw/workspace-search/skills/multi-search-engine/scripts

# 设置 API Key
export EXA_API_KEY="your-exa-key"

# 运行搜索
python3 policy_search.py "科技创新政策" 10
```

参数说明：
- 第一个参数：搜索关键词
- 第二个参数（可选）：每个省份返回的结果数量，默认 5

### 方法二：在对话中调用

你可以通过对话让我帮你搜索，只需说：

```
帮我搜索"科技创新政策"，每个省份返回10条结果
```
