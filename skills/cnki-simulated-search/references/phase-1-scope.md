# Phase 1 Scope

## Included in phase 1

- CNKI search-page capture
- search-result list extraction
- basic detail-page/abstract-page extraction
- Excel output for list and detail pages

## Not included in phase 1

- guaranteed full-text PDF download
- login bypass
- captcha bypass
- permission circumvention
- guaranteed access to all detail pages

- `~/Desktop/cnki-h5/`
- `~/Desktop/cnki-output/`
