# Search Expansion Planner

## 目标

把一句自然语言搜索需求拆成结构化检索计划：
- 提取年份
- 提取地区
- 提取主题
- 对主题做领域方向扩展
- 输出为桌面 CSV 表格，供后续搜索或人工检查使用

默认规则：
- 未给年份时，默认当前年份
- 未给地区时，默认湖北省
- 当前版本重点支持“农业政策”主题扩展

## 执行方式

优先直接运行脚本：

```bash
python3 scripts/generate_expansion_plan.py "帮我搜索2026年湖北省所有农业政策"
```

也可不给参数，脚本会提示输入：

```bash
python3 scripts/generate_expansion_plan.py
```

## 输出位置

默认输出到桌面：

`~/Desktop/搜索配置文件夹/current-expansion-plan.csv`

若目录不存在，先自动创建。

## 工作流

### 1. 解析输入

从用户原话中识别：
- `year`：优先提取四位年份；未提取到则使用当前年份
- `province`：优先匹配省/市/自治区名称；未提取到则使用“湖北省”
- `policy_need`：保留原始需求文本
- `theme`：从需求里抽取政策主题，当前重点识别“农业政策”

若只识别到“农业”“农业相关政策”“三农政策”“乡村振兴政策”等，也统一归并到农业主题扩展。

### 2. 主题扩展

把笼统主题扩成更可检索的细分方向。当前农业主题默认扩展为：
- 粮食安全政策
- 高标准农田建设政策
- 种业振兴政策
- 农机装备政策
- 智慧农业政策
- 农产品加工政策
- 农业产业化政策
- 设施农业政策
- 绿色农业政策
- 生态循环农业政策
- 农业科技创新政策
- 农业数字化政策
- 农业经营主体扶持政策
- 家庭农场政策
- 农民合作社政策
- 龙头企业扶持政策
- 农业金融支持政策
- 农业保险政策
- 农产品流通政策
- 冷链物流政策
- 乡村振兴政策
- 农村土地改革政策
- 耕地保护政策
- 农业人才政策
- 农业补贴政策

如需支持更多主题（例如新能源、文旅、博物馆），把扩展模板补进 `references/theme_templates.json`。

### 3. 生成表格

输出 CSV 至桌面，建议字段保持稳定：
- 年份
- 地区
- 原始需求
- 标准主题
- 扩展方向
- 搜索词
- 状态
- 执行备注

其中：
- `状态` 默认写 `待执行`
- `执行备注` 默认留空
- `搜索词` 由 `年份 + 地区 + 扩展方向` 组成

## 资源

### scripts/
- `generate_expansion_plan.py`：解析输入、扩展主题、生成 CSV

### references/
- `theme_templates.json`：主题到扩展方向的模板映射；优先在这里维护扩展词，不要把大段模板直接堆进 SKILL.md

## 迭代约定

- 新增主题时，优先修改 `references/theme_templates.json`
- 若用户要求“默认地区”或“默认年份”变化，再调整脚本默认值
- 若后续要接入主搜索脚本，保持 CSV 字段名稳定，避免执行层频繁改动
