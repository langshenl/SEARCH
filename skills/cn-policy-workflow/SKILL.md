# 中国政策搜索完整工作流

## 核心流程

```
用户输入搜索词
     ↓
判断是否需要扩展（触发判断）
     ↓
执行扩展或不扩展
     ↓
调用exa-web-search-free搜索（每个扩展词20条）
     ↓
合并去重（保证≥20条）
     ↓
保存Excel到 ~/Desktop/政策文件夹/
```

## 第一步：触发判断

**需要扩展的条件（同时满足）：**
1. 搜索词包含省份/城市名称（如：湖北、上海、北京、武汉、深圳）
2. 搜索词包含"政策"一词
3. 搜索词是宽泛类型（无具体政策类型）

**不需要扩展的示例：**
- "2026湖北新能源汽车政策" → 包含具体政策类型"新能源汽车"，❌ 不扩展
- "2026人工智能政策" → 无省份/城市，❌ 不扩展
- "2026湖北产业政策" → 已有具体类型"产业"，❌ 不扩展

**需要扩展的示例：**
- "2026湖北政策" → 宽泛，✅ 扩展
- "2026年上海政策" → 宽泛，✅ 扩展
- "2026武汉政策" → 宽泛，✅ 扩展

## 第三步：执行搜索

使用 exa-web-search-free 对每个扩展词执行搜索：

```bash
mcporter call 'exa.web_search_exa(query: "2026年湖北财政政策 site:gov.cn", numResults: 25)'
```

**搜索策略：**
- 每个扩展词搜 **numResults=25**（确保有20+条可用结果）
- 自动添加 `site:gov.cn` 限定政府官网
- 搜索时使用 `type: "deep"` 获得更全面的结果

**搜索参数优化：**
```python
search_config = {
    "numResults": 25,      # 每扩展词25条
    "type": "deep",        # 深度搜索
    "min_results_per_keyword": 20,  # 每扩展词最少20条
    "capture_published": True  # 捕获发布时间
}
```

**Exa返回结果字段：**
- `Title`：标题
- `URL`：原文地址
- `Published`：发布时间（如"2026-01-15T00:00:00.000Z"）
- `Highlights`：内容摘要
- `Author`：来源

## 第四步：合并去重

**去重规则：**
1. **URL去重**：URL完全相同只保留一条
2. **标题去重**：标题相似度>80%去重
3. **内容去重**：正文内容相似>90%去重

**结果保证：**
- 每个扩展词保证 ≥20条 独立结果
- 扩展词之间去重后总结果 ≥20条
- 如果总结果不足20条，继续搜索其他扩展词补充

## 第五步：数据质量保证

**准确性验证：**
1. 来源必须是 gov.cn 域名（政府官网）
2. 标题必须包含政策关键词
3. 正文必须包含有效政策内容
4. 过滤无效/错误/重复结果

**真实性验证：**
1. URL必须是有效链接
2. 发布时间必须是有效日期
3. 来源必须可识别
4. 正文长度必须>50字

## 第六步：输出Excel

**输出路径：** `~/Desktop/政策文件夹/{省份}_{关键词}_{时间戳}.xlsx`

**Excel格式：**

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 标题 | 正文 | 摘要 | 来源地方 | 原文地址 | 发布时间 |

**字段说明：**
- **标题**：政策文件/文章的完整标题
- **正文**：正文摘要（100-300字），包含政策核心内容
- **摘要**：简短摘要（50字内），一句话说明政策要点
- **来源地方**：来源网站名称（如"湖北省人民政府"）
- **原文地址**：原始页面完整URL
- **发布时间**：政策发布日期（如"2026-01-15"）

## 第七步：结果展示

**终端输出：**
```
## 搜索结果（共 X 条，来源：gov.cn）

### 1. [标题](原文链接)
**来源**：xxx | **摘要**：xxx

正文摘要...

---
```

## 搜索量计算

假设：20个扩展词 × 每词25条搜索 → 500条原始数据 → 去重后约150-200条可用数据

**最低保证：每扩展词20条 × 扩展词数量 = 足够的数据量**

## 注意事项

- **结果数量**：每个扩展词搜索保证≥20条，不足则重新搜索
- **来源可信**：必须是 gov.cn 域名结果
- **去重标准**：URL>标题>内容 三级去重
- **保存路径**：固定为 `~/Desktop/政策文件夹/`
- **准确性优先**：宁可少搜几个扩展词，也要保证每条数据准确
