### 1. 创建政策搜索技能系列

**cn-policy-workflow** - 中国政策搜索完整工作流
- 自动判断是否需要扩展搜索词
- 调用exa高级搜索（带日期过滤）
- **省份过滤：根据URL识别省份，不在搜索时限定域名**
- 每个扩展词保证20+条结果
- 输出字段：标题、正文、摘要、来源地方、原文地址、发布时间

**cn-policy-search-expand** - 政策搜索扩展词
- 触发条件：省份/城市 + "政策" + 年份
- 支持20种政策类型扩展

**cn-policy-multi-search** - 多引擎政策搜索
- 31省市政府官网域名清单
- 搜索策略：政府官网优先

### 2. 安装 exa-web-search-free

- 通过 SkillHub 安装：`skillhub install exa-web-search-free`
- 依赖 mcporter CLI（已安装）
- 配置：`mcporter config add exa-full "https://mcp.exa.ai/mcp?tools=web_search_exa,web_search_advanced_exa,..."`
- **注意：免费MCP有速率限制**

### 3. 关键问题修复

**问题1：搜索到其他省份的政策**
- 原因：`includeDomains: ["gov.cn"]` 太宽泛
- 解决：不限定域名，在结果中根据URL过滤省份

**问题2：Exa速率限制**
- 错误信息："You've hit Exa's free MCP rate limit"
- 解决：等待恢复，或考虑其他方案（Tavily）

**问题3：JSON解析**
- Exa返回大JSON时非贪婪匹配失效
- 解决：使用括号计数法匹配数组边界
